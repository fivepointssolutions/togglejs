    Namespaces

    • Toggle

    Classes

    • Toggle.CheckboxBehavior

      Allows a the selection of a checkbox to toggle an element or group of elements on and off. Just set the rel attribute to "toggle[id1,id2,...]" on the checkbox.

    • Toggle.LinkBehavior

      Allows a link to toggle the display of another element or array of elements on and off. Just set the rel attribute to "toggle[id1,id2,...]" on the link and the href of the link to the ID of the first element ("#id1").

    • Toggle.RadioGroupBehavior

      Allows you to toggle elements based on the selection of a group of radio buttons. Just set the rel attribute to "toggle[id1,id2,...]" on each radio button. Radio buttons must be grouped inside a containing element to which the behavior is applied.

    • Toggle.SelectBehavior

      Allows you to toggle elements based on the selection of a combo box. Just set the rel attribute to "toggle[id1,id2,...]" on the each select option.
