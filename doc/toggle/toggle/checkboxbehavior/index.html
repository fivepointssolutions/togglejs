      class Toggle.CheckboxBehavior

      Description

      Allows a the selection of a checkbox to toggle an element or group of elements on and off. Just set the rel attribute to "toggle[id1,id2,...]" on the checkbox.

      Options

      • invert: When set to true the associated element is hidden when checked.
      • effect: This option specifies the effect that should be used when toggling. The default is "slide", but it can also be set to "blind", "appear", or "none".
