    class method Toggle.wrapElement 

    Toggle.wrapElement(element) → Element

    Utility function. Wraps element with a div of class "toggle_wrapper" unless one already exists. Returns the "toggle_wrapper" for given element. This is necessary because effects only work properly on elements that do not have padding, borders, or margin.
